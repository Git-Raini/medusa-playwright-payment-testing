# Medusa Payment Workflows

## Guest Checkout

Browse Products
→ Add Product to Cart
→ Checkout
→ Shipping Address
→ Payment
→ Order Confirmation

---

## Registered User Checkout

Login
→ Browse Products
→ Add Product to Cart
→ Checkout
→ Payment
→ Order Confirmation

---

## Refund Workflow

Order Created
→ Refund Request
→ Partial Refund
→ Full Refund

---

## Payment Failure Workflow

Checkout
→ Invalid Card
→ Payment Declined
→ Error Validation
